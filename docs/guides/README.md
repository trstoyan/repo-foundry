# Guides

Guides explain how to use Repo Foundry patterns in practice.

- `idea-to-plan.md`: move from a raw idea to an agent-operable plan.
- `adopting-existing-project.md`: apply Repo Foundry to a project that already
  has entrypoints, validation, docs, and runtime rules.
- `contributing-lessons.md`: turn observed project lessons into safe upstream
  discovery notes, profiles, guides, templates, or validation checks.
- `prompts.md`: copyable prompts for planning, implementation, and review.
