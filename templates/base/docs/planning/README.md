# Planning

Use this directory to turn an idea into a safe, agent-operable plan before
implementation starts.

Recommended flow:

1. Fill out `idea-brief.md`.
2. Draft `project-plan.md`.
3. Identify risk with `risk-profile.md`.
4. Create the first small implementation slices with `slice-template.md`.
5. Summarize agent context in `agent-context.md`.

The goal is to let AI agents help expand the idea while keeping scope,
validation, risks, and open questions visible.
