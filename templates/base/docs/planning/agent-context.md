## Agent Rules

- Read `AGENTS.md` first.
- Treat `docs/planning/` as planning truth until implementation truth exists.
- Do not start implementation from a broad idea. Start from an accepted slice.
- Keep changes scoped to the slice.
- Builder summaries are not evidence.
- Report validation and remaining risks.

- "List open questions that block implementation."

- "Create acceptance criteria and validation for this slice."
