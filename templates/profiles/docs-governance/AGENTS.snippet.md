## Documentation Authority

This project treats documentation as part of the implementation.

- `README.md` describes project identity, quick start, and navigation.
- `AGENTS.md` defines the agent operating contract.
- `DEVELOPMENT.md` defines contributor workflow and validation commands.
- `SYSTEM_STATUS.md` describes current implemented truth.
- `CHANGELOG.md` records chronological history.
- `docs/decisions/` records durable decisions.
- `docs/runbooks/` records operational procedures.

If code, tests, current-status docs, and operator-visible behavior disagree,
the change is incomplete. Resolve documentation drift before reporting
completion.
